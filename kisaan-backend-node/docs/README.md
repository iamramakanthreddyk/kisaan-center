# Kisaan Backend Node.js - API Documentation Index

This folder contains API and schema documentation for each resource.

- Add a new markdown file for each resource (e.g., `auth-api.md`, `users-api.md`, `products-api.md`).
- Document endpoints, request/response schemas, and example payloads.
- Update this index as new APIs are added.

## Example
- `auth-api.md` - Login, registration, JWT, password reset
- `users-api.md` - User CRUD, roles, permissions
- `products-api.md` - Product CRUD, categories
- `db-schema.md` - Database schema and migration notes
